# SIO Review — Approve or Reject Suggestions

Review pending suggestions generated by `/sio-suggest`. Each suggestion includes a proposed rule, confidence score, pattern context, and target file.

## Triggers (natural language)

- "Review my suggestions"
- "Show me what SIO recommends"
- "What suggestions are pending?"
- "Let me review the improvement rules"
- "Go through the suggestions"

## Execution

```bash
#!/bin/bash
set -e

echo "=== SIO Review: Pending Suggestions ==="
echo ""
sio suggest-review
```

## What Each Suggestion Contains

For each suggestion, the review shows:

| Field | Description |
|---|---|
| **ID** | Suggestion number for approve/reject commands |
| **Description** | What error pattern this addresses |
| **Confidence** | How sure SIO is (based on error count, session count, examples) |
| **Target file** | Where the rule would be written (CLAUDE.md, skill files, hooks) |
| **Proposed change** | The actual rule text that would be added |
| **Pattern** | The error cluster that triggered this suggestion |

## Approve / Reject Commands

```bash
# Approve a suggestion (promotes to ground truth)
sio approve <ID> --note "prevents repeated Bash retries"

# Reject a suggestion with a reason
sio reject <ID> --note "too generic, need more data"
```

## Review Workflow

1. `/sio-suggest` generates suggestions from mined error patterns
2. `/sio-review` shows each pending suggestion for your decision
3. Approve the good ones, reject the bad ones
4. `/sio-apply` writes approved suggestions to their target files

## After Review

- **Approved** → Ready for `/sio-apply` to write to CLAUDE.md or target file
- **Rejected** → Archived with your note, won't resurface
- **Skipped** → Stays in queue for next review session

## Tips

- High confidence + high error count = strong signal, approve
- Low confidence = DSPy wasn't sure, inspect the proposed rule carefully
- If a suggestion is too generic ("be more careful") → reject, the pattern needs more examples
- Approved suggestions become ground truth for future DSPy optimization
